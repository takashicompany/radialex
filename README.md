# Radialex

40%サイズのキーボードです。
放射状のキーレイアウトは手を自然な体勢のまま置くことが可能で、指を最短距離で動かすことで文字を打つことが可能です。
キーボードの中央部にはあなたの象徴となるキーやキーキャップを配置することができます。
LEDのバックライトで輝かせること可能です。
また、MXソケットによるキースイッチの付替えも可能です。
